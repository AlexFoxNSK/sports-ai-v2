# ⚽ Sports AI v2.0

**Система ставок на тоталы (Over/Under 2.5) с XGBoost.**

## 📊 Результаты Paper Trading (Этап 15)

| Метрика | Значение |
|---------|----------|
| **Банк** | 10 000 → 11 345₽ |
| **ROI** | **+13.45%** |
| **Проходимость** | **65.9%** |
| **Ставок** | 88 (из 2 856 туров) |
| **Средний EV** | +20.7% |

## 🧠 Модель

| Параметр | Значение |
|----------|----------|
| Алгоритм | XGBoost (200 деревьев, max_depth=5) |
| Признаков | 18 (позиции, голы, Over%, H2H, отдых) |
| Калибровка | Ручная поправка на основе binning-анализа |
| Целевая | Over/Under 2.5 |

## 🎯 Стратегия ставок

| Параметр | Значение |
|----------|----------|
| Порог ТБ 2.5 | p > 0.75 (калиброванная) |
| Порог ТМ 2.5 | p < 0.35 (калиброванная) |
| EV фильтр | > 5% |
| Стейк | 2% от банка |
