# ⚽ Sports AI v2.0

**Система ставок на тоталы (Over/Under 2.5) с XGBoost.**

## 🎯 Цель
Построить прибыльную модель ставок на тоталы, начиная с чистого листа.

## 📅 План разработки
| Этап | Задача | Статус |
|------|--------|--------|
| 1 | XGBoost OU 2.5 + 20% порог | ⬜ |
| 2 | Признаки (голы, xG, форма) | ⬜ |
| 3 | Валидация (Backtest + CLV) | ⬜ |
| 4 | Стратегия (Kelly + фильтры) | ⬜ |
| 5 | Paper Trading | ⬜ |

## 🆕 Отличия от v1.x
- XGBoost вместо LightGBM
- Тоталы вместо 1X2
- 20% порог на старт каждого сезона
- Честная валидация без подгонки
